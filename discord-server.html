Hoe maak ik een eigen Discord-server aan

Terug naar Kennisbank

Stappen

  1. Open de Discord-app of website.
  2. Klik op het **+**-teken in de linker zijbalk.
  3. Kies **Maak een server**.
  4. Geef je server een naam en upload een serverafbeelding (optioneel).
  5. Klik op **Maken** en je server is klaar!

Je kunt nu kanalen maken en leden uitnodigen naar je server.
